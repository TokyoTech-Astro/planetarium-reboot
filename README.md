# TokyoTech-Astro Planetarium

東工大天文研究部のプラネタリウムプロジェクト。

新刊活動に向けて再始動中！

## Description

光学式の投影機とエアドームの自作プラネタリウム。

数年前まで新刊活動や工大祭で使用していた。コロナや工大祭の中止の影響でノウハウが失われつつある。

### Components

- ドーム

- 構造球
  - 恒星投影筒
  - 星座絵投影筒
  - 一等星投影筒

- 昼光
 
- 流星投影機

- スピーカー

- 制御システム

- 番組
